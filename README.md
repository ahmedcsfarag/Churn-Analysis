# Customer Churn Analysis

## Project Overview
This project focuses on analyzing customer behavior to identify the key factors that lead customers to leave the service (Customer Churn).

The analysis uses a dataset containing customer demographics, subscription details, usage patterns, and customer service interactions. The goal is to uncover insights that help businesses understand churn drivers and improve customer retention strategies.

---

## Tool Used
- Microsoft Excel

---

## Dataset Description
The dataset includes several variables related to customer behavior such as:

- Age
- State
- Contract type
- Data usage
- Customer service interactions
- Churn status
- Reasons for leaving the service

---

## Project Objectives
The analysis aims to:

- Identify customer segments with the highest churn rate
- Analyze the main reasons behind customer churn
- Compare the impact of competitors versus pricing and service quality
- Explore the relationship between customer usage behavior and churn probability

---

## Dashboard
An interactive dashboard was created using Microsoft Excel to visualize churn patterns across multiple dimensions such as:

- Age groups
- Customer location (State)
- Contract type
- Data usage
- Reasons for churn

The dashboard helps decision-makers quickly explore trends and identify key churn drivers.

---

## Key Insights
The analysis helps highlight:

- Customer groups that are more likely to churn
- The most common reasons behind customer churn
- The relationship between usage behavior and churn
- Opportunities to improve customer retention strategies

---

## Business Value
This analysis provides valuable insights that can help businesses:

- Improve customer experience
- Reduce churn rate
- Strengthen customer retention strategies
